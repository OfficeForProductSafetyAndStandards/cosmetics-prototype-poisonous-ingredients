How to prepare a cosmetic product notification

To submit a product notification you can either use data from an EU notification or give the information manually.

Using an EU notification file

If your products have been notified in the EU, you can use the data from your EU notification. You will need to:

  • export your data as a ZIP file for each notification – this needs to be the same format as exported from CPNP
  • upload the ZIP files to this service
  • Files exported from CPNP do not contain all the information required for a notification.

You can upload 100 files at a time.

Manually

If you don’t have EU notification files, you need to manually enter information about the product.
